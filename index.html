Idea

NDN micro-forward is an in-browser forwarder implemented in Javascript. The idea is to use one of the forwarder as a relay to outside world while other devices on the same network use multicast to retrieve content from the relay. This is useful where Internet connectivity is not very good.

Tasks

  • Protocol to select a gateway
  • Stand up a NDNFS server for serving test content
  • Implement a in-browser content store
  • Implement pipeline for retrieving content
  • Multicast configuration scripts for local network
  • Test for Firefox and Chrome
  • Test for Linux and Mac
  • Stream video through a set of micro-forwarders
  • Easy installation script
